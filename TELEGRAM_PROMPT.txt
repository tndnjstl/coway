아래 작업을 순서대로 전부 진행해줘. 중간에 묻지 말고 바로 실행해.

[1] 필수 도구 확인
brew --version, git --version, python3 --version, php --version 각각 실행해서 확인해. 없으면 brew로 설치해.

[2] 프로젝트 클론
mkdir -p ~/DEV/php && cd ~/DEV/php && git clone https://github.com/tndnjstl/coway.git
완료되면 git -C ~/DEV/php/coway log --oneline -3 결과 알려줘.

[3] sites.xml 생성 (FTP 배포 설정 - gitignore 파일이라 클론 안 됨)
~/DEV/php/coway/sites.xml 을 아래 내용으로 생성해줘:

<?xml version="1.0" encoding="UTF-8"?>
<FileZilla3>
    <Servers>
        <Server>
            <Host>112.175.185.138</Host>
            <Port>21</Port>
            <User>tndnjstl</User>
            <Pass encoding="base64">cmtkZGt3bDEh</Pass>
        </Server>
    </Servers>
</FileZilla3>

[4] FTP 배포 테스트
cd ~/DEV/php/coway && python3 ftp_deploy.py
"배포 완료!" 메시지 나오면 성공.

[5] 로컬 개발 서버 스크립트 생성
~/DEV/php/coway/start_dev.sh 파일 생성:
#!/bin/bash
cd ~/DEV/php/coway/html
php -S localhost:8080

그리고 chmod +x ~/DEV/php/coway/start_dev.sh 실행.
로컬 서버는 http://localhost:8080 으로 접속.

[6] 최종 상태 확인
아래 명령어 실행하고 결과 정리해서 알려줘:
- git -C ~/DEV/php/coway log --oneline -1
- php --version | head -1
- python3 --version
- ls ~/DEV/php/coway/sites.xml
- ls ~/DEV/php/coway/app/helpers/common_db.php

- 코드 수정 후 항상 git commit + python3 ftp_deploy.py 배포까지 자동으로
- DB 쿼리는 $db_local->real_escape_string() 필수
- 응답은 한국어로
- 작업 디렉토리: ~/DEV/php/coway
